# Azure DevOps Configuration

This directory can contain Azure DevOps-specific configuration files.

## What Can Go Here

### Pull Request Templates

**File:** `.azuredevops/pull_request_template.md`

Example:
```markdown
## Description
Brief description of changes

## Type of Change
- [ ] New feature
- [ ] Bug fix
- [ ] Infrastructure change
- [ ] Documentation update

## Testing
- [ ] Validated Bicep templates
- [ ] Tested in dev environment
- [ ] Reviewed what-if analysis

## Checklist
- [ ] Code follows project standards
- [ ] Documentation updated
- [ ] All tests pass
```

### Work Item Templates

Custom work item templates for bugs, features, etc.

### Pipeline Extensions

Custom tasks or extensions configuration.
